# LiveContainer Sources

Unofficial AltStore-compatible sources intended for use with LiveContainer.

## StikDebug

Official project repository:

https://github.com/StephenDev0/StikDebug

Source URL:

```text
https://raw.githubusercontent.com/intraducine/livecontainer-sources/main/stikdebug.json
```

The source is checked once per day by GitHub Actions. When a new official StikDebug GitHub release is published, the workflow updates the version, build number, release date, download URL, file size, minimum iOS version, and release notes in `stikdebug.json`.

This repository and source are unofficial and are not affiliated with the StikDebug developer, AltStore, or LiveContainer.
